# WarpMaster

![WarpMaster Logo](/Other/wm-logo.png)

Программа для Minecraft-OpenComputers, позволяющая удобно работать с ядром корабля из мода WarpDrive.

## Требования и установка

Для работы программы на компьютере требуется установленная MineOS ( https://github.com/IgorTimofeev/MineOS ). Для установки в дополнение к этому требуется интернет-карта.

Для установки WarpMaster войдите в режим консоли (в MineOS левый верхний угол - вернуться в Shell) и выполните команду:
`pastebin run W3ZHAArT`

После успешной установки вернитесь обратно в MineOS командой:
`OS`
Ярлык программы появится на рабочем столе.

Для правильной работы программа может отключать автозагрузку MineOS при старте компьютера. Если это происходит, используйте команду
`OS`
чтобы запустить систему.
